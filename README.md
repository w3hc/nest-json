# Nest.js JSON

Testing Nest.js in combination with a JSON-based database.

- Swagger UI: https://nest-json.jcloud-ver-jpe.ik-server.com/api
- UI: https://github.com/w3hc/nest-json-ui

## Install

```bash
pnpm i
```

## Run

```bash
pnpm start
```

## Test

```bash
$ pnpm test
```

## Versions

- pnpm v8.7.5
- node v20.9.0
